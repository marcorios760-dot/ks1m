# Claude rules for this repository

## Git

- **Never push without explicit user confirmation.** After committing, report what was committed and wait for the user to say "push it" (or similar) before running `git push`.
- Prefer editing existing files over creating new ones.
- Do not amend published commits.

## README / docs

- Keep the README in sync with any script changes.
- Do not add horizontal rules (`---`) between sections.

## Structure

- Images go in the `img/` directory.
- rinkhals-conf is the backup of the ks1m rinkhals custom settings
